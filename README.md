# Задание для рефакторинга: Интернет-магазин

## Описание проекта

Этот проект представляет собой простой интернет-магазин, написанный на React с использованием Redux Toolkit. **Код содержит типичные антипаттерны**, которые необходимо исправить.

## Цель задания

Улучшить качество, читаемость и поддерживаемость кода за счёт:
- устранения антипаттернов;
- более чёткого разделения ответственности;
- упрощения архитектуры Redux Store.

## Функциональность

Приложение включает:
- Каталог товаров с фильтрацией и поиском
- Корзину покупок
- Добавление/удаление товаров
- Изменение количества товаров
- Оформление заказа

* рефакторинг стилей и верстки оцениваться не будет

## Основные изменения

### Redux Store и архитектура
- Redux Store разделён по ответственности:
  - `appSlice` - продукты, пользователь, loading, error;
  - `cartSlice` - состояние корзины.
- Удалены все вычисляемые значения из store, оставлены только источники истины.
- Производные данные вынесены в memoized selectors (`createSelector`).

### Dispatch и side effects
- Убран `dispatch({ type })`, используются action creators.
- Асинхронная логика вынесена из компонентов в `createAsyncThunk`.

### UI и компоненты
- `App` разделён на `Header`, `ProductList`, `Cart`.
- Компоненты отвечают только за UI и пользовательские действия.

### Обработка ошибок
- Добавлено состояние `error` в store.
- Реализован компонент `ErrorMessage` с возможностью повторной загрузки данных.

## Инструкции по запуску

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект:
```bash
npm run dev
```

3. Откройте браузер по адресу `http://localhost:5173`
